Allows LIST, GET, PUT, APPEND, DELETE and MKDIR of files on a remote FTP server.

Inputs

filename string
the path/name of the file on the remote FTP server.
localFilename string | buffer
the path/name of the file on the local machine.

Outputs

  1. Standard output (List)
    payload array
    array of the file/folders in the user's default ftp folder
  2. Standard output (Get, Put, Append, Delete and Mkdir)
    payload string
    Message that processing is complete.
  3. Standard error
    payload string
    Error message.

Details

msg.filename is the path/name of the file on the remote FTP server

msg.localFilename is the path/name or buffer of the file on the local machine

NOTE: the local machine is where NR is running - this might not be the machine where your browser connects to Node Red.

GET uses msg.filename and msg.localFilename

PUT uses msg.filename and msg.localFilename

APPEND uses msg.filename and msg.localFilename

DELETE uses only msg.filename

LIST uses msg.filename. Returns an array of the file/folders in the user's default ftp folder, containing type, name, sticky, rights(an object), acl, owner, group, size and date.

MKDIR uses msg.filename to specify the directory name. Recursive addition is allowed.

You can pass msg.filename and msg.localFilename to this node, however values entered in the node will take precedence.
