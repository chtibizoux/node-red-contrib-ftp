Configuration for a connection to an remote FTP server.


Host is The hostname or IP address of the remote FTP server. Default: localhost

Port is The port of the remote FTP server. Default: 21

User is Username for authentication. Default: anonymous

Password is Password for authentication. Required input.

Connection Timeout is How long (in milliseconds) to wait for the control connection to be established. Default: 10000

PASV Timeout is How long (in milliseconds) to wait for a PASV data connection to be established. Default: 10000

Keepalive is How often (in milliseconds) to send a 'dummy' (NOOP) command to keep the connection alive. Default: 10000

Data connection encryption is Set to true for both control and data connection encryption. Default: false
